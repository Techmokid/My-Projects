Raw basic data:
 - 100 trillion connections (Assume worst case scenario 128 trillion connections just in case)
 - 86 billion neurons (Assume worst case scenario 100 billion just in case)
 - 1k connections per neuron average
 - 15k connections per neuron max



Useful info to keep in mind:
 - We'll assume instead of 100 trillion connections, 128 trillion to take into account human brain variation
 - We'll assume instead of 86 billion neurons, 100 billion to take into account human brain variation
 - Indexing of 128 trillion connections requires 47 bits per connection (This bit range is capable of a maximum 140,737,488,355,328 connections if necessary) <- UNUSED
 - Indexing of 100 billion neurons requires 37 bits per neuron (This bit range is capable of a maximum 137,438,953,472 neurons if necessary)
 - 15k connections per neuron max needs 14 bits for indexing minimum



"Synapse connection" comprises of the following:
 - 37 bits - Index to the target neuron ID/index (What this neuron connects to)
 - 32 bits - Neuron propagation delay. To account for signal propagation delay between longer distances. Counted in picoseconds
 - 16 bits - Amount of myelination on the connection, from 0 to 65,535
 - 16 bits - Synapse weight
 - 08 bits - Plasticity value
 - 02 bits - Neurotransmitter class
 - 01 bits - Excitatory vs Inhibitory synapse
TOTAL = 112 bits/connection = 14 Bytes/connection



"Neuron" comprises of the following:
 - 37 bits - Neuron ID/index (Correlates to "Synapse connection" ID for simplicity)
 - 32 bits - Neuron soma X coordinate (For building 3D visual neural models if desired)
 - 32 bits - Neuron soma Y coordinate (For building 3D visual neural models if desired)
 - 32 bits - Neuron soma Z coordinate (For building 3D visual neural models if desired)
 - 16 bits - Neuron type
 - 16 bits - Neuron firing threshold (float measured in Volts)
 - 16 bits - Neuron refractory period
 - 15 bits - Number of Synapse definitions following this neuron definition
 - 08 bits - Activation function parameters
 - 04 bits - Spare bits not yet assigned (Used for forward compatibility and aligning bits)
TOTAL = 208 bits/neuron = 26 Bytes/neuron

We have taken a data shortcut with synapse connections.
All connections are assumed that they start at the current neuron in the data to save space.
E.g. "Synapse connection 4" under Neuron 2 will automatically be assumed coming from neuron 2 and going out to whichever neuron it's 37 bit "Next neuron ID" points to.
This saves a heap of space by removing 37 bits from the synapse definition
